方案优势

  1. 内容集中管理
    所有Markdown内容在<script>标签中集中维护，与HTML结构解耦。

  2. 精准定位
    通过data-markdown-source和data-markdown-target属性精确控制插入位置。

  3. 扩展性强
    新增视频或内容区块时，只需复制结构并修改ID即可，无需修改JS代码。

  4. 避免转义问题
    使用<script type="text/markdown">天然避免HTML特殊字符冲突。
