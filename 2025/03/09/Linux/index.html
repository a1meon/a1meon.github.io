一、基本命令

1. ls命令-列出文件

格式：ls [-a -l -h] [linux路径]

-a：all 列出全部文件（包括隐藏的文件/文件夹）

-l：以列表形式展示（竖向，展示更多信息）

-h：以易于阅读的形式展示文件大小（显示单位）

2. cd命令-切换工作目录

格式：cd [linux路径]

cd 无选项，如果不写参数，表示回到用户的home目录

3. pwd命令-查看当前工作目录

格式：pwd

4. mkdir命令-创建新的目录

格式：mkdir [-p] Linux路径

-p：选项可选，表示自动创建不存在的父目录，适用于创建连续多层级的目录

5. touch命令-创建文件

格式：touch [Linux路径]

6. cat命令-查看文件内容

格式：cat [Linux路径]

7. more命令-分页查看文件内容

  • cat是直接将文件内容全部显示，more可以分页显示，适用于文件内容较多的情况

格式：more [Linux路径]

按Q退出

8. cp命令-复制文件或文件夹

格式：cp [-r] 参数1 参数2

-r：可选，用于复制文件夹

参数1：表示被移动的文件或文件夹

参数2：表示要移动去的地方

9. mv命令-移动文件或文件夹

格式：mv 参数1 参数2

参数1：表示被移动的文件或文件夹

参数2：表示要移动去的地方

10. rm命令-删除文件或文件夹

格式：rm [-r -f] 参数1 参数2 ...... 参数n

-r：用于删除文件夹

-f：force，表示强制删除（不会弹出提示信息）

参数1 参数2 …… 参数n表示要删除的文件

11. which命令-查找命令的程序文件

格式：which [命令]

12. find命令-查找指定文件/按文件大小查找文件

查找指定文件：find 起始路径 -name “被查找文件名“

按文件大小查找文件：find 起始路径 -size +|-n[kMG]

find支持通配符，test*、*test、*test*

+、-表示大于和小于，n表示大小数字，kMG表示大小单位,kb、MB、GB

13. grep命令-从文件中通过关键字过滤文件行

格式： grep [-n] 关键字 文件路径

-n：表示在结果中显示匹配的行的行号

关键字：必填，表示过滤的关键字，带有空格或其他特殊符号，建议”“将关键字包起来

文件路径：必填，被过滤的文件路径

14. wc命令-数量统计

格式：wc [-c -m -l -w] 文件路径

-c：统计bytes数量

-m：统计字符数量

-l：统计行数

-w：统计单词数量

文件路径，被统计的文件

15. 管道符 - |

将管道符左边命令的结果作为右边命令的输入

格式：command1 | command2

例：

查找一个日志文件中包含特定关键词的行，并计算其数量

cat logfile.txt | grep "关键词" | wc -l

16. echo - 在命令行内输出指定内容

格式：echo 输出的内容

反引号 - `

被``包围内容会作为命令执行

重定向符 - >和>>

  • > ：将左侧命令的结果，覆盖写入到符号右侧文件
  • >> ：将左侧命令的结果，追加吸入符号右侧

17.tail命令 - 查看文件尾部内容

格式：tail [-f -num] Linux路径

-f：表示持续跟踪

-num：表示查看尾部多少行

Linux路径，表示被跟踪的文件路径

18.vi 编辑器

三种工作模式

命令模式（Command mode）：vi编辑器默认进入命令模式，在输入模式下按esc返回

输入模式（Insert mode）：i、a、o进入

底线命令模式（Last line mode）：冒号（:）进入

二、用户

1. su命令-切换用户

格式：su [-] [用户名]

-：是可选的，表示是否在切换用户后加载环境变量，建议带上

用户名，表示要切换的用户，用户名也可以省略，省略表示切换到root

切换用户后，可以通过exit命令退回上一个用户，也可以使用快捷键：ctrl + d

2. sudo命令-临时以root身份执行其他命令

格式：sudo 其他命令

我们需要为普通用户配置sudo认证

3. 用户组管理

创建用户组：groupadd 用户组名

删除用户组：droupdel 用户组名

4. 用户管理

创建用户：useradd [-g -d] 用户名

-g：指定用户的组，如不指定会自动创建用户组并加入

-d：指定用户HOME路径，如不指定，HOME默认在/home/用户名

删除用户：userdel [-r] 用户名

-r：删除用户的home目录，不适用-r，删除时home目录保留

查看用户所属组：id [用户名]

用户名为被查看的用户，如果不提供则查看自身

修改用户所属组：usermod -aG 用户组 用户名

将指定用户加入指定用户组

查看当前系统中用户：getent passwd 查看当前系统中的用户

查看当前系统中用户组：getent group 查看当前系统中的用户组

5. 查看权限控制信息

-：文件 d：文件夹 l：软链接

r：读权限 w：写权限 x：执行权限

例：drwxr-xr-x，d表示文件夹，rwx表示所属用户权限，r-x表示所属用户组权限，r-x表示其他用户权限

6. chmod命令-修改文件或文件夹的权限

只能是文件或文件夹的所属用户或root用户有权限修改

格式：chmod [-R] 权限 文件或文件夹

-R：对文件夹内的全部内容应用同样的操作

例：

chmod u = rwx,g=rx,o=x hello.txt，将文件权限修改为rwxr-x–x

chmod -R u=rwx,g=rx,o=x test，将文件夹test及文件夹内的全部内容权限修改为rwxr-x–x

权限可以用3位数字来表示

0：- - - 1：- -x 2：-w- 3：-wx

4：r- - 5：r-x 6：rw- 7：rwx

7. chown命令-修改文件或文件夹的所属用户、用户组

普通用户无法修改所属为其他用户或组，所以此命令只适用于root用户执行

格式：chown [-R] [用户] [:] [用户组] 文件或文件夹

-R：同chmod，对文件夹内全部内容应用相同规则

用户：修改所属用户

用户组：修改所属用户组

: ：用户分隔用户和用户组

例：

chmod root hello.txt 将hello.txt所属用户修改为root

chown :root hello.txt 将hello.txt所属用户修改为root

chown root:demo hello.txt 将hello.txt所属用户修改为root，用户组修改为demo

chown -R root test 将文件夹test的所属用户修改为root并对文件夹内全部内容应用同样规则

三、Linux使用技巧

1. 各类小技巧

强制停止程序运行（或命令输入错误，退出当前输入）：ctrl + c

退出账户的登录（退出某些的特定程序的专属页面）：ctrl + d

查看历史输入过的命令：history

匹配历史命令：ctrl + r

光标移动快捷键：

ctrl + a跳到命令开头

ctrl + e跳到命令结尾

ctrl + 键盘左键，向左跳一个单词

ctrl + 键盘右键，向右跳一个单词

清空终端内容：ctrl + l 或 clear命令

2. 软件安装

yum安装软件

yum：rpm包软件管理器，用于自动化安装配置linux软件，并可以自动解决依赖问题。

yum命令

格式：yum [-y] [install | remove | search] 软件名称

-y：自动确认，无需手动确认安装或卸载过程

install：安装

remove：卸载

search：搜索

3. systemctl命令

格式：systemctl start | stop | status | enable | disable 服务名

start启动 stop关闭 status查看状态 enable开机自启 disable关闭开机自启

系统内置的服务都可以通过systemctl控制，例如：

NetworkManager，主网络服务

network，副网络服务

firewalld，防火墙服务

sshd，ssh服务

4. ln命令-创建软连接

格式：ln -s 参数1 参数2

-s ：创建软链接

参数1：被链接的文件或文件夹

参数2：要链接去的目的地

5. date命令

格式：date [-d] [+格式化字符串]

-d：按给定的字符串显示日期

格式化字符串：通过特定的字符串标记，控制显示的日期格式

%Y：年

%y：年份后两位数字（00-99）

%m：月份

%d：日

%H：小时

%M：分钟

%S：秒

%s：自1970-01-01 00:00:00 UTC 到现在的秒数

修改Linux时区

需要使用root权限

ntp程序

通过ntp自动校准系统时间

6. IP地址和主机名

ifconfig命令-查看本机的ip地址

hostname 查看主机名

hostnamectl set-hostname 主机名 （需要root）

8. 网络请求和下载

使用ping命令检查服务器是否可联通

格式：ping [-c num] ip或主机名

-c：检查的次数，不使用将无限次持续检查

ip或主机名，被检查的服务器的ip地址或主机名地址

使用wget命令下载文件

格式：wget [-b] url

-b：可选，后台下载，会将日志写在当前目录wget-log文件

url，下载链接

使用curl命令发起网络请求

语法：curl [-0] url

-O，用于下载文件，当url是下载链接时，可以使用此选项保存文件

url，要发起请求的网络地址

9. 端口

查看主机端口占用：nmap 被查看的ip地址

nmap不是系统自带命令，需要先安装：yum -y install nmap

查看指定端口占用：netstat -anp | grep 端口号

netstat也不是系统自带命令，需要先安装：yum -y install net-tools

10. 进程管理

查看进程：ps [-e -f]

-e，显示出全部的进程

-f，以完全格式化的形式展示信息（展示全部信息）

字段名 描述
UID 进程所属的用户ID
PID 进程的进程号ID
PPID 进程的父ID（启用此进程的其他进程）
C 此进程的CPU占用率（百分比）
STIME 进程的启动时间
TTY 启动此进程的终端序号，如显示？表示非终端启动
TIME 进程占用CPU的时间
CMD 进程对应的名称或启动路径或启动命令

查看指定进程：ps -ef | grep tail

关闭进程：kill -9 进程ID

-9，表示强制关闭进程

11.主机状态

查看系统资源占用：top

-默认每5秒刷新一次，直接输入top即可，按q或ctrl + c退出

top命令内容详解：http://linux.51yip.com/search/toptop

top命令支持的选项：

-p，只显示某个进程的信息

-d，设置刷新时间，默认是5s

-c，显示产生进程的完整命令，默认是进程名

-n，指定刷新次数，比如top -n 3，刷新输出3次后退后

-b，以非交互非全屏模式运行，以批次的方式执行top，一般配合-n指定输出几次统计信息，将输出重定向到指定文件，比如top -b -n 3 > /tmp/top.tmp

-i，不显示任何闲置（idle）或无用（zombie）的进程

-u，查找特定用户启动的进程

每一份支持都如同星星般闪耀✨
